# [7] 차트 및 데이터 시각화 컬러 규칙서

본 문서는 와이앤아처 통합 Works 플랫폼의 대시보드(통계, 현황 리포트 등)에서 쓰이는 차트 및 데이터 시각화 요소의 **시각적 일관성과 정보 판독성, 웹 접근성**을 보장하기 위해 표준 시각화 컬러 및 디자인 사양을 정의합니다.

---

## 1. 시각화 설계 원칙 (Visualization Principles)

*   **높은 구별성 (Distinctiveness)**: 다중 계열 차트에서 인접한 데이터 계열 간의 색상이 명확히 구별되도록 명도와 색상 차이를 확보합니다.
*   **시각적 균형 (Visual Balance)**: 특정 데이터 계열이 불필요하게 강조되지 않도록 채도와 밝기가 균등한 톤을 가진 컬러 팔레트를 채택합니다.
*   **접근성 보장 (Color Accessibility)**: 적록색약 등 색약 사용자도 데이터의 경계를 흐름이나 톤 차이로 식별할 수 있도록 형태 보조선(패턴, 점선, 마커)의 활용 규칙을 병행합니다.

---

## 2. 범주형 차트 컬러 팔레트 (Chart Categorical Colors)

동일 화면 내에서 다수의 비교군 데이터를 표현할 때, 순차적으로 적용하는 **8종의 전역 표준 차트 컬러**입니다.

| 순서 | 토큰명 | 대표 색상 | 색상 코드 (HEX) | 사용 가이드라인 |
| :---: | :--- | :--- | :--- | :--- |
| 1 | **`chart.1`** | **Blue (기본색)** | `#2563EB` | 단일 차트의 단독 데이터 또는 가장 중요한 첫 번째 계열 |
| 2 | **`chart.2`** | **Green** | `#16A34A` | 성공, 긍정, 안정적 성장 또는 두 번째 비교 데이터 계열 |
| 3 | **`chart.3`** | **Amber (주황)** | `#D97706` | 주의, 경고, 혹은 세 번째 비교 데이터 계열 |
| 4 | **`chart.4`** | **Purple** | `#7C3AED` | 인구통계, 네 번째 비교 계열 또는 서브 데이터 강조 |
| 5 | **`chart.5`** | **Teal (청록)** | `#0D9488` | 친환경, 보육 기수 통계, 다섯 번째 비교 계열 |
| 6 | **`chart.6`** | **Pink** | `#DB2777` | 여성 비율, 감성 통계, 여섯 번째 비교 계열 |
| 7 | **`chart.7`** | **Indigo (인디고)** | `#4F46E5` | 기술/인프라 분야 통계, 일곱 번째 비교 계열 |
| 8 | **`chart.8`** | **Gray** | `#4B5563` | 기타(Other) 데이터 계열, 전체 평균선 또는 기준값선 |

*(※ 8개 이상의 다중 계열이 필요한 경우, 위 팔레트의 투명도를 60%로 낮춘 톤이나 패턴을 병행하여 디자인 노이즈를 방지합니다.)*

---

## 3. 차트 배경 요소 및 그리드 라인 명세

차트 영역의 레이아웃 구성 요소는 데이터 판독에 방해가 되지 않도록 은은하고 깔끔하게 마감합니다.

*   **축 및 그리드 라인 (Grid Lines)**:
    *   **X/Y축 메인 가이드라인**: `1px` 실선 | `#E5E5E5` (`gray.200`).
    *   **배경 가이드선 (Grid lines)**: `1px` 점선 (`stroke-dasharray: 4 4`) | `#F0F0F0` (`gray.100`).
*   **축 레이블 텍스트 (Axis Labels)**:
    *   **글자 규격**: `12px` (`text-xs`) | Regular.
    *   **컬러**: `#737373` (`gray.500`).
    *   **배치**: 데이터와 겹치지 않도록 최소 `8px` 이상의 여백 마진을 확보합니다.

---

## 4. 범례(Legend) 및 툴팁(Tooltip) 스타일 규격

### 4.1 차트 범례 (Legend)
*   **마커 형태 및 크기**:
    *   **선형/영역형 차트**: `8px * 8px` 크기의 완전 원형(`rounded-full`) 마커를 범례 글씨 좌측에 배치합니다.
    *   **막대형 차트**: `10px * 10px` 크기의 미세 라운드 사각형(`radius.sm`) 마커를 사용합니다.
*   **텍스트 규격**: `12px` (`text-xs`) | Medium | `#515151` (`gray.600`) | 마커와의 간격 `6px`.

### 4.2 차트 마우스오버 툴팁 (Chart Tooltip)
차트의 특정 데이터 포인트에 마우스를 올렸을 때 나타나는 팝오버 명세입니다.

```
+---------------------------+
| 2026년 2분기 AC 실적      |  <- 헤더 (12px, gray.500)
| ■ 포트폴리오 A : 24.5 억  |  <- 데이터 행 (14px, gray.800)
| ■ 포트폴리오 B : 12.0 억  |
+---------------------------+
```

*   **배경 및 외곽선**: `#FFFFFF` 단색 배경 | `border border-gray-200` (`#E5E5E5`) 테두리 | 모서리 `6px` (`radius.md`).
*   **그림자 효과**: `shadow.dropdown` (`0 4px 12px rgba(0, 0, 0, 0.08)`) 적용.
*   **내부 패딩**: `12px` (`p-3`) 전체 패딩 적용.
*   **컨텐츠 레이아웃**:
    *   **툴팁 헤더 (기준점 명칭 등)**: `12px` (`text-xs`) | Bold | `#737373` (`gray.500`) | 하단 마진 `8px` (`mb-2`).
    *   **데이터 행**: `14px` (`text-sm`) | Regular | `#1F1F1F` (`gray.900`) | 가로 `flex items-center gap-2` 형태로 좌측에 컬러 마커 배치.
