# [3-4-6] AC 멘토링 상호 평가 피드백 루프 상세 요건서

본 문서는 1:1 멘토링 세션 종료 후 스타트업과 전문가 멘토 간에 발생하는 양방향 평가 피드백 폼의 데이터 스키마 및 이 데이터가 전사 허브(`HUB`) 랭킹 보드와 스타트업 상세 뷰에 실시간 연동되는 규칙을 정의합니다.

---

## 1. 양방향 평가 피드백 흐름도

```
[ 멘토링 세션 완료 ] 
  │
  ├─ 1. 스타트업 ➔ 멘토 평가 ───► [ 만족도 점수(1~5) & 의견 ] ───► [ HUB 전문가 랭킹 (실시간 정렬) ]
  │
  └─ 2. 멘토 ➔ 스타트업 평가 ───► [ 5대 성장 지표 (각 1~5) ] ────► [ NETWORKS 스타트업 상세 (성장 데이터) ]
```

---

## 2. 양방향 평가지 세부 필드 명세 (Guest Viewport 연동)

모든 평가는 외부 이해관계자용 모바일/웹 뷰포트(`GUEST`)에서 번거로운 절차 없이 터치 조작으로 원활히 제출할 수 있도록 최적화된 필드를 제공해야 합니다.

### 2.1 스타트업 ➔ 전문가 만족도 평가 폼 (Startup ➔ Mentor Evaluation)
* **제출 시점**: 멘토링 세션 상태가 `completed`로 변경된 직후 스타트업 모바일 Guest 화면에 팝업 또는 알림을 통해 노출됩니다.
* **입력 필드 구성**:
  1. **멘토링 종합 만족도**: 1~5점 별점 척도 선택 (필수값, 기본값 없음, 터치 영역 `48px * 48px` 이상 확보).
  2. **세부 만족도 피드백**: 자유 서술형 textarea 필드 (선택 입력, 최대 500자 제한, 모바일 리사이징 반응형 마진 패딩 적용).
* **데이터 귀속**:
  - 해당 데이터는 `mentor_satisfaction_records` 테이블에 저장되며, AC 담당자의 해당 사업 상세 뷰 내 '멘토링 기록 상세 패널'에서 조회할 수 있습니다.

### 2.2 전문가 ➔ 스타트업 진단 평가 폼 (Mentor ➔ Startup Diagnostic Evaluation)
전문가 멘토가 멘토링 세션 진행 중 또는 직후에 스타트업의 역량과 사업성을 정밀 진단하는 평가 양식입니다.

* **제출 시점**: 멘토링 완료 후 전문가 Guest 스케줄 보드 내 대상 스타트업 상세 카드 하단에서 [상담일지 및 평가지 작성] 클릭 시 진입합니다.
* **입력 필드 구성 (5대 정량지표)**:
  1. **기술성 (Technology)**: 기술적 경쟁력 및 독창성 (1~5점 별점).
  2. **사업 BM 적절성 (Business Model)**: 수익 모델 구체성 및 시장성 (1~5점 별점).
  3. **신뢰도 (Credibility)**: 팀 역량 및 준비성, 실행 의지 (1~5점 별점).
  4. **협업 잠재력 (Collaboration Potential)**: 대기업/협력사 및 와이앤아처와의 협업 가능성 (1~5점 별점).
  5. **매칭 성사율 (Matching Feasibility)**: 향후 후속 투자 유치 또는 M&A 딜 성사 가능성 (1~5점 별점).
  6. **종합 자문 의견**: 스타트업에 대한 핵심 개선 조언 및 면담 요약 텍스트 필드 (필수 입력, 최소 20자 이상).

---

## 3. 타 워크스페이스 실시간 연동 규칙

### 3.1 HUB 전문가 만족도 랭킹 보드 실시간 반영
스타트업이 전문가에게 준 평점 데이터는 백그라운드 스케줄러에 의해 실시간으로 평균 계산되어 전사 데이터 센터에 정렬 반영됩니다.

* **평균 만족도 산출 수식**:
  - `(대상 전문가 UUID의 mentor_satisfaction_records 내 score 총합) / (평가 완료된 총 세션 수)`
* **HUB 연동**:
  - 이 산출 수식을 거쳐 계산된 값은 `HUB` 자문단 만족도 랭킹 보드([3_1_workspace_hub.md](./3_1_workspace_hub.md))에 실시간 반영되어, **평균 평점 내림차순(Highest to Lowest)**으로 전문가 목록이 자동 정렬 렌더링됩니다.

### 3.2 NETWORKS 스타트업 상세 성장 지표 연동
전문가가 제출한 5대 성장 지표 데이터는 단순한 일지 보관을 넘어 스타트업의 고유 성장 역사 데이터로 상속됩니다.

* **데이터 상속 규칙**:
  - 전문가가 평가를 제출하는 즉시, 대상 스타트업의 `NETWORKS` UUID로 평가 데이터가 연동 매핑됩니다.
  - `NETWORKS` 스타트업 상세 페이지 내 '성장 지표 및 피드백 히스토리' 영역에 **레이더 차트(Radar Chart)** 형태로 5대 정량지표 평균값이 누적 갱신되어 표출됩니다.
  - 시간 경과에 따른 스타트업의 성장을 분석하는 백데이터로 영구 활용됩니다.
