# [3-4-7] AC 커스터마이즈 행사 및 활동 메모 상세 요건서

본 문서는 보육 프로그램 운영 중 발생하는 비정형 보육 행사(수시 워크숍, 간담회, 스타트업 네트워킹 데이 등)의 세부 진행 성과와 현장 사진을 텍스트 메모 형태로 가볍게 기록 및 아카이빙하는 '커스터마이즈 행사 활동 메모(Custom Activity Log) 탭'의 요건을 정의합니다.

---

## 1. 활동 메모 및 사진 타임라인 UI 구성

이 탭은 연도/월별 시간 순서에 따라 카드가 나열되는 타임라인 피드 형태로 구성되며, 각 카드 내에는 현장 사진 슬라이더와 텍스트 기록이 렌더링됩니다.

## 2. 세부 기능 및 비즈니스 요건

### 2.1 행사 활동 기록 작성 및 편집
* **작성/편집 모달 폼 필드**:
  1. **행사 타이틀**: 필수 입력 text 필드 (최대 100자).
  2. **행사 일자**: 필수 입력 데이트 피커 (과거 또는 당일 날짜만 선택 가능, 미래 날짜 불가).
  3. **참석 스타트업 지정**: 현재 보육 프로그램에 매핑되어 있는 스타트업 목록 중 참석한 대상들을 다중 선택(체크박스 또는 Multi-select)하여 매핑합니다.
  4. **활동 내용 기술**: 마크다운을 지원하는 textarea 에디터 영역 (최대 2,000자 제한, 진행 내용 및 성과 요약 기록).
  5. **이미지/현장 사진 업로드**:
     - 드래그 앤 드롭 파일 업로드 컴포넌트(Dropzone).
     - **업로드 제약 규칙**: 이미지 파일 포맷(`.jpg`, `.png`, `.webp` 등)만 업로드 가능하며, 1회 등록 시 최대 5장까지 제한합니다. 개별 파일 크기는 최대 10MB로 제안하며, 초과 시 오류 메시지와 함께 업로드를 차단합니다.
     - 업로드된 이미지는 서버 측 이미지 리사이저(Image Resizer)를 거쳐 웹 최적화 해상도(가로 최대 1080px) 및 WebP 포맷으로 압축 저장되어 로딩 성능을 보장합니다.

### 2.2 목록형 타임라인 피드
* **렌더링 구조**:
  - 작성된 활동 기록 카드가 최신 등록 일자 순(내림차순, Descending Order)으로 타임라인 라인에 매핑되어 세로 피드(Feed) 형태로 나열됩니다.
  - 개별 카드는 참석 스타트업 칩(Chip), 작성자명, 마크다운 렌더링 텍스트, 이미지 캐러셀(Carousel, 사진이 2장 이상일 경우 좌우 넘기기 버튼 제공)을 기본 구성으로 노출합니다.

### 2.3 스타트업 상세 뷰(`NETWORKS`)의 활동 아카이브 연동
본 기능은 AC 워크스페이스뿐만 아니라, 피보육 대상인 스타트업의 자체 성장 이력 포트폴리오를 관리하기 위해 `NETWORKS` 워크스페이스와 연동됩니다.

* **연동 및 표출 규칙**:
  - 관리자가 활동 기록 폼에서 '참석 스타트업'으로 지정한 특정 스타트업의 고유 UUID를 감지합니다.
  - 해당 스타트업의 `NETWORKS` 상세 뷰([3_3_workspace_networks.md](./3_3_workspace_networks.md)) 진입 시, **'보육 프로그램 참가 이력 및 활동 아카이브'** 영역에 본 활동 기록 카드(행사명, 일자, 현장 사진, 본문 요약)가 실시간으로 연동되어 동일하게 렌더링됩니다.
  - 스타트업은 본인이 참여한 모든 활동 메모를 본인 Guest 프로필에서 일목요연하게 회고할 수 있습니다.
  - 이 연동은 **조회(Read) 전용**이며, 내용 수정 및 사진 삭제는 오직 등록한 AC 워크스페이스 내 활동메모 탭에서만 수행할 수 있습니다.
