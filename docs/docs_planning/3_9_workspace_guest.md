# [3-9] GUEST (외부 파트너용 서브도메인 앱) 상세 기능 요건서

본 문서는 사외 이해관계자인 스타트업, 외부 전문가, 임시 게스트가 WORKS 내부 앱과 분리된 별도 서브도메인 기반 앱에서 번거로운 회원가입 없이 간편하고 안전하게 플랫폼 서비스에 접속하고 협업 태스크를 완료할 수 있도록 화면 요건과 비즈니스 흐름을 정의합니다.

---

## 1. 주요 화면 및 기능 명세

### 1.1 스타트업 Guest 대시보드
* **보육 프로그램 상세 타임라인 (Timeline View)**
  * **기능**: 본인이 속한 보육 사업 기수의 전체 공식 일정(오리엔테이션, 교육 세션, 중간 심사, 데모데이 등)을 달력 및 타임라인 그래프로 직관적으로 확인합니다.
* **1:1 미팅/멘토링 예약 시스템 (Booking Console)**
  * **기능**: 보육 프로그램에서 배정해 준 가용 전문가 풀을 조회하고 멘토링 시간대를 신청합니다.
  * **조작 흐름**:
    1. 전문가 카드(소속, 분야, 사진 약력 포함) 목록을 조회합니다.
    2. 원하는 전문가를 선택하여 비어 있는 시간대 슬롯을 확인합니다.
    3. 원하는 시간대 슬롯을 터치하여 간편 예약 신청을 완료합니다.
* **멘토 만족도 평가지 제출 화면 (Evaluation Form)**
  * **기능**: 멘토 D전문가와 미팅이 종료된 직후 활성화되는 만족도 설문 입력 창입니다.
  * **주요 항목**: 멘토링 만족도 별점(1~5점), 텍스트 피드백 의견(자유 기술 서술형).

### 1.2 전문가 Guest 대시보드
* **멘토링 스케줄 보드 (Schedule Board)**
  * **기능**: 나에게 배정되거나 예약 확정된 스타트업 목록, 날짜, 시간 정보를 목록 형태로 일목요연하게 제공합니다.
* **스타트업 상세 요약 카드 (Startup Summary Card)**
  * **기능**: 미팅 진행 전, 대상 스타트업의 비즈니스 모델(BM) 한줄 소개, 대표자명, 홈페이지 주소, 스타트업이 제출한 PDF 사업소개서를 터치하여 즉시 열어볼 수 있는 프리뷰 카드를 제공합니다.
* **상담일지 및 평가지 제출 화면 (Counseling Form)**
  * **기능**: 멘토링 세션 완료 후 전문가가 스타트업을 진단하는 정량/정성 평가 리포트 폼입니다.
  * **평가 지표**: 기술성, 사업 BM 적절성, 신뢰도, 협업 잠재력, 매칭 성사율 (각 1~5점 척도 별점 선택) 및 종합 코멘트 조언 텍스트 필드.

### 1.3 임시 게스트 뷰 (Temporary Guest View)
* **일회성 링크 전용 뷰포트 (One-time Access Port)**
  * **기능**: 관리자가 발급한 고유 보안 접속 주소(난수 해시값 포함)로 진입한 임시 게스트에게 제공되는 격리 화면입니다.
  * **화면 연출**: 일반적인 메뉴바나 내비게이션 요소가 전혀 렌더링되지 않으며, 사전에 임직원이 발급 시 지정한 **지정된 특정 정보 영역만 화면 단독 렌더링**됩니다.
  * **만료 연출**: 기한이 만료되거나 세션 창을 닫은 후 다시 진입하면 "만료된 접속 주소입니다"라는 안내 페이지로 자동 리다이렉트되어 정보 보안을 보장합니다.

## 2. 모바일 기기 사용성 최적화 정책

* **터치 영역 및 버튼 최소 높이**: 모바일 화면에서 오터치 없이 원활하게 예약을 진행하고 평점을 부여할 수 있도록, 모든 별점 별 터치 영역 및 확인 버튼의 최소 크기를 `48px * 48px` 이상으로 확보합니다.
* **텍스트 영역 키보드 반응형 리사이징**: 모바일 가로모드나 좁은 뷰포트에서 의견 서술을 위해 키보드가 활성화되었을 때 입력 폼이 화면 밖으로 밀려 잘리지 않도록 화면 크기가 자동으로 줄어드는 키보드 마진 패딩 설정을 기획에 명시합니다.
